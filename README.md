# Baruch CIS 3630

**CIS 3630 - Principles of Web Design**

Website that allows consumers to review and purchase solar related products.

<https://katsute.github.io/Baruch-CIS-3630>

### Installation

 1. Download [NodeJS](https://nodejs.org/).
 2. `cd project`
 3. `npm i`
